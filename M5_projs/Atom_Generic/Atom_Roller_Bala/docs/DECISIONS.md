## 000 — 采用 in-repo `docs/` + `CLAUDE.md` 作为本地持久化与接手体系

- **决策**:任务的需求 / 设计 / 状态 / 决策 / 操作全部落盘到项目内 `docs/`,根目录 `CLAUDE.md` 作薄入口(树状,详细协议在 `docs/HANDOFF.md`);全部入 git。

## 001 — 控制律先 PID、后 LQR
- **日期**:2026-06-20
- **决策**:平衡控制先用 PID 实现(把系统跑起来、能稳),项目基本能运转后再评估加入 LQR。

- **影响**:DESIGN §6;RollerCAN 的 I2C 控制模式(速度/位置/电流)随 PID 实现确定。
