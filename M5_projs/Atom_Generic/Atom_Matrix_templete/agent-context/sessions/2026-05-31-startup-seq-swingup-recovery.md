   - 实测(run_recover2)：面内翻到 84°/lat=−2 → 第2个击杀方向把它**磕回 B(+30°)，自救成功**。
   - 对比 lat=−38°(三维翻滚)救不回 → **垂直角是可救性关键判别**。
