# 003 — IMU 姿态滤波：单轴 pitch 互补滤波（陀螺主导）

- 日期：2026-05-30
- 状态：已采纳（取代 dir_test 中原 accel 低通；补充 [001](001-imu-readout.md) 的纯加速度算角）

## 背景
电机方向辨识实测中，飞轮一转动，**纯加速度算出的 pitch 就失真**，多次造成防翻保底**误触发断电**。
需判断：现有读数/算角里有没有合适滤波？应移植哪种成熟方案？它带来多少延迟？

## 关键实测证据（来自 tools/serial.log，2026-05-29 那次跑）
静止 WAIT 段（150 帧）：
- 加速度 pitch：均值 30.46°，**峰峰仅 0.57°**（静止时加速度算角很干净）。
- 陀螺零偏：gx≈-2.07、**gy≈+0.10（pitch 轴）**、gz≈+0.55 °/s；gy 噪声 σ≈0.11。

误触发段 SETTLE（电机已停、飞轮 535→468rpm 滑行，125ms）：
| 现象 | 数值 |
|------|------|
| 加速度 pitch | 28.8 → **42.8°**（凭空 +14° 暴冲） |
| 原 LPF(α=0.6) 后 | 仍冲到 **37.1°** → 越 34° 危险线，误断电 |
| pitch 轴陀螺 gy | ±5°/s **零均值振动**，扣偏置后**积分净转角 = 0.03°** |

结论：机体**根本没动**。那 42.8° 是飞轮滑行的**线加速度伪象**——是低频实信号，
**低通滤波去不掉**（低通只压高频噪声）。陀螺才是动态下唯一可信的角速度源。

## 决策
在 IMU 模块实现**单轴 pitch 互补滤波**，陀螺主导、加速度做秒级慢校正：

```
fused = α·(fused + (gy - gyBias)·dt) + (1-α)·accelPitch,   α = 0.98
```
- pitch 轴 = **gy**（由几何 + 数据确认：pitch 在 x-z 平面内，绕 Y 轴）。
- 运行前**静止现标陀螺零偏**（采 ~1s 取均值，扣 gy 偏置）——每次上电略变。
- API：`imuCalibrateGyro/imuGyroBias` + `imuFusionInit/imuFusionStepPitch/imuFusedPitch`（见 src/imu.*）。
- dir_test 的看门狗/判向信号由原 accel-LPF 改为该融合值（CSV `pf` 列含义同步更新为"陀螺融合角"）。

## 延迟分析（用户要求量化）
- **原 accel 低通（α=0.6, dt=20ms）**：时间常数 τ=-dt/ln α≈**39ms**（~2 帧群延迟），
  且**根本没解决线加速度伪象**——双输——既滞后又误判。
- **互补滤波（α=0.98, dt=20ms）**：陀螺路径对真实转动**零附加滞后**（输出 = 上次 + 陀螺·dt，
  实时跟随，延迟≈采样周期 20ms 本身）；加速度校正时间常数 τ=α/(1-α)·dt≈**980ms**，
  远低于机动动态频率 → 对控制相关信号**不引入滞后**。
  净动态延迟 ≈ 一个采样周期，**优于低通**。
- 代价：陀螺会缓慢漂移，由加速度慢校正消除（静止几秒即收敛回真值）；机动期(亚秒)漂移可忽略（实测 0.03°/125ms）。

## 考虑过的替代方案
- **纯加速度 + 低通**（原方案）— 去不掉线加速度伪象，已实测误触发，弃。
- **1D 卡尔曼**（经典 MPU6050 平衡车用）— 更优、能估陀螺偏置，但计算重、调参多；
  单轴此场景互补滤波已足够，先用互补，必要时再升级。
- **Madgwick / Mahony**（无人机 AHRS 常用）— 四元数全 3D，对单轴摆是过度设计。
- **滑动平均 / 中值**— 只压高频、带群延迟，同样去不掉低频伪象，弃。
- **片上 DLPF + 提采样率抗混叠**— 正交手段，后续可叠加（当前 50Hz，平衡阶段或需提到 200Hz）。

## 影响 / 代价
- 好处：动态下角度可信，防翻保底不再被飞轮振动/滑行误触发；为后续起摆/平衡控制打下可信状态基础。
- 成本：运行前必须静止标定陀螺（~1s）；需按真实 dt 积分（已用 millis 时间戳）。
- 待实测确认：α=0.98 是否合适（机动更剧烈时可能需调）；50Hz 采样下陀螺振动混叠是否需叠加片上 DLPF。

## 实测验证（2026-05-30，见 sessions/2026-05-30-dir-test-with-filter.md）
- ✅ **有效**：转动期间 accel 角峰峰 62.9°，融合 pf 峰峰仅 4.7°(σ0.63)，伪象抑制~93%，方向辨识首次完整跑完、0 误触发。
- ✅ α=0.98 当前够用。
- ⚠️ 证实高转速(220mA/~600rpm)陀螺振动混叠 ±40°/s（近零均值、互补滤波能扛），高频平衡控制阶段需开片上 DLPF / 提 ODR。
- ✅ 证实陀螺零偏每次上电会漂（gy +0.10→+0.40°/s），**现标策略正确**。
