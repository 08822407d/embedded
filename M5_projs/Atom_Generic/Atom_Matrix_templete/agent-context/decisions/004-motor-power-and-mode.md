# 004 — 电机供电受限下的控制方式探索（速度模式 / 秋千蓄能 / CAN 升压）

- 日期：2026-05-30
- 状态：进行中（实测驱动）

## 背景
[003 起跳表征](003-imu-filtering.md) 之后实测：电流模式下命令 1200mA 只得 ~85mA 实际电流、
飞轮顶到 ~600rpm、机体只前进 ~1.5°，**无法起跳**。诊断 Vin≈4.5V 且满载不掉压 → **电机仅靠 Grove 5V、电压受限**。
用户确认：**Grove 供电确实只有 5V**。

## 用户指示（决策）
1. **先换速度模式**控制 RollerCAN，对比哪种模式扭矩更大 / 能观测到更大摆幅。
2. 再考虑**模仿秋千多次摆动蓄能**（共振泵能：每半周顺着摆动方向给力，逐周累积幅度）。
3. **安全**：系统**横滚角最好不要超过 42°**。
   - 当前代码阈值更保守：面内(pitch)危险 ±34°、侧向(lateral)15° —— 均在 42° 内，符合"不超 42"。暂不放宽。
4. 若仍不行：查资料确认 **RollerCAN 经 CAN 接口供 12-15V 是否能得到更高扭矩/转速**（升压供电路线）。

## 理由 / 思路
- 电流模式自限电流（命令 1200 只给 85mA），速度模式控制器会在 setSpeedMaxCurrent 内**施加最大努力**逼近目标转速，
  动态加速段可能榨出更大瞬时力矩 → 更大反作用冲量 → 更大摆幅。值得实测对比。
- 但 4.5V 下飞轮 back-EMF 顶速 ~600rpm 是物理墙，速度模式也越不过；秋千泵能靠"多次小冲量累积"绕过单次冲量上限。
- CAN + 12-15V 若可行，是根本解（电压上去，力矩/转速都上去）。

## 影响 / 代价
- 模式在 init 一次性选定，**运行中不切**（见 [002](002-motor-control-strategy.md)）；故速度模式 = 单独固件路径。
- 速度模式下安全响应须用速度指令（setSpeed(0)/断输出），**不能**用电流模式的反向电流恢复。

## 实测结论（2026-05-30，tools/run_speed.log / run_speed2.log）
- **速度模式不比电流模式强**：飞轮同样顶到 ~580rpm，机体融合 pf 仅在 30±1.5° 抖动、**根本不脱阱**，与电流模式一致。
  ⇒ **~600rpm 飞轮上限是 4.5V 供电的硬墙，与控制模式无关**，速度模式越不过。
- **新问题：高转速下 gy 振动 ±40~83°/s 严重混叠**，连**陀螺融合角**都会被同号连击短暂顶出 3~4°假尖峰，
  造成**误触发保底**（run2 报 pitch=34 实为假尖峰，full 轨迹 pf 从未真到 34）。
  ⇒ 高 authority 控制/平衡前必须开 **MPU6886 片上 DLPF** 或**提采样率**抗混叠。
- **侧向融合有效**：raw 侧向角曾假报 22°，加 gx 互补融合后稳在 ~-1.8°，证实那是线加速度伪象（同 pitch）。
- **秋千泵能预判**：机体单次仅响应 ~1.5° 且静止在棱上阻尼大，泵能可能很难起效；但未实证，存疑。
- **结论：问题是供电，不是控制策略**。下一步走 **CAN + 12-15V 升压供电**。

## 供电规格查证结论（m5-docs，2026-05-30）✅ 用户假设成立
- RollerCAN **CAN(XT30) 口支持 6–16V DC**；**Grove 仅 5V**。电机 D3504 200KV。
- **16V vs 5V：力矩 0.065 vs 0.021 N·m（≈3×），电流 927 vs 350mA**。→ **接 12–16V 必然大幅提升力矩/转速**。
- 过压：>18V 报 E:1 停机；**上限 16V 勿超**。
- **行动建议**：XT30 接 **12–15V DC**（留余量、勿近 18V），Grove 继续接做 I²C 通信。
  供电到位后，回到**电流模式起跳表征**重测（预期此时能脱阱/起摆）。
- 详细规格已记入 [protocols/rollercan-i2c.md](../protocols/rollercan-i2c.md#供电)。

## 12V 上电台架实测（2026-05-30，tools/run_bench.log）
- 用户给 XT30 接 ~12V（Grove 5V **未断开**，两路并存）。实测 **Vin=12.65V，err=0，无过压/异常**——并联无问题。
- **空载台架能力（不做起跳，机体受硬线约束）**：
  - **最大飞轮转速 ≈ 1900rpm**（4.5V 时 ~600rpm → **≈3×**；理论空载 200KV×12.6≈2500rpm）。
  - 空转峰值电流 ≈ 240mA（各档 200~1200mA 命令都 ~230mA）——**空载被 back-EMF 限流**，非受限；
    带载（真起跳时）电流会高得多（规格 16V 带载 ~900mA）。
- **结论**：升压到 12V，反作用轮可用角动量 ≈3×，authority 足够，起跳/起摆应可行。
- ⚠️ 现实障碍：**XT30 供电线太硬，会约束/干扰机体摆动**——做带机体运动的起跳/起摆前需先解决走线（柔性线/理线/滑环）。
