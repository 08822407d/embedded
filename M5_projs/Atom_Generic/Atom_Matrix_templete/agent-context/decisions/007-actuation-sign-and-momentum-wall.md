## 一、信号可靠性分级(关键教训)

| 信号 | 何时可信 | 何时**不可信** |
|---|---|---|
| **飞轮转速 spd(电机编码器)** | **始终可信** | —— |
| 加速度 pitch | 飞轮≈0 且落定后 | 飞轮转动时被反作用/振动污染(顶到37–85°假象) |
| 陀螺 gy | 低转速 | **高转速混叠失真**(system-model §4：220mA/600rpm 即 ±30–46°/s;本季高速急停段积分出 ±190~290° 荒谬值) |
| 融合 pf | 低速 | 高速被加速度污染带跑(本季"接管−17°/最接近−0.8°"多半是污染假象) |

⇒ **判机体运动方向：高速段只信飞轮编码器(spd) + 落定后的加速度 pitch(飞轮停)。绝不用高速段的 pf/gy 判方向。**

## 二、铁律(模型推导,不需再测)

`I_b·θ̈ = m·g·l·sinθ − τ`，`τ=Kt·I`(电机给飞轮)，反作用 **−τ 驱动机体**；`τ=I_w·ω̇_w`。
- 牛顿第三定律：**机体受力矩 = −(飞轮力矩) ⇒ 机体被推方向恒与飞轮"角加速度方向"相反**。冲量版：`I_b·Δθ̇ ≈ −I_w·Δω_w`(快动作、重力可忽略)。
- ⇒ **机体净推向 = 与飞轮净 Δω_w 相反**，可直接从 spd 列读，不必看机体。
- **突然加速 vs 高速急停**：要把机体推同方向就要飞轮角加速度同号；直接加速=飞轮从0往一方向加速，急停=先把飞轮转到**相反方向**再刹回0。**两方案飞轮转向相反、机体被推同方向**——必然,可推导。

## 三、绝对符号(用可靠信号重新定，**推翻历史记录**)

铁证(编码器 + 落定加速度,均可信)：
- 直接 **+电流** → 飞轮 spd 23→**+1863rpm**(往正加速)。⇒ Kt>0：+电流→飞轮正转。
- 直接 **−400mA** → 飞轮 **−1742rpm**(负)，机体落定 **+84°(第三面/朝B外)**。⇒ −电流→飞轮负→Δω_w<0→机体 +θ 外翻。✓ 自洽。
- ⇒ **+电流 → 飞轮正加速 → 机体 −θ(从B朝平衡)**。即 **从B朝平衡 = 正电流；从A朝平衡 = 负电流**：`朝平衡电流符号 = (pitch>0)? +1 : −1`。
- **2026-05-30 记的"负电流朝平衡"作废**(当时5V/低速、或电流符号约定不同;与编码器铁证冲突,以编码器为准)。

由此推导两种起跳的正确符号(从B为例,目标 Δω_w>0)：
- **直接加速·朝平衡(B)= 正电流**。`directSwingUp/Rescue` 已是 `(p0>0)?+1:−1` ✓ 正确。
- **蓄能急停·朝平衡(B)= 蓄能负电流(飞轮转负)→ 急停正电流(刹回0,Δω_w>0)**。
  ⇒ brakeKickCore 应传 **kickSign=(p0>0)?+1:−1**。**本季 brakeKickCore 各调用方(cmd 3/4、g、m、v)都传成了 (p0>0)?−1:+1 = 反的** ⇒ 蓄能急停一直把机体推**外翻到第三面**，而"最接近平衡−0.8°/接管−17°"都是 pf 污染假象。这解释了**为什么几乎每次起跳都倒向第三面**。

## 四、动量预算墙(结构性,软件不可破)

- 反作用轮**只能把机体稳在 θ=0**(重力力矩为零)；稳在任何非零角需飞轮持续加速=无限转速,不可能。⇒ 起跳必须把机体**以低速送到 θ≈0** 才接得住。
- 一切冲量/授权 ∝ `I_w·Δω_w`，而 **ω_max 被电压限死 ~1900rpm(12V)**。⇒ 起跳冲量、第三面回收、平衡授权**同一堵墙**。
- 实测：PID 接管后 **0.2s 飞轮即撞 ~1900rpm 饱和**，之后零反作用力矩、零授权；断电时该动量一泄把机体甩翻。
- ⇒ 根治：**抬供电电压(↑ω_max=↑动量预算) + 单轴约束夹具(侧向不可控)**。

## 五、正确的测试方法(以后照此)

1. 判方向只用**飞轮编码器 spd** + **落定加速度 pitch**；高速段 pf/gy 仅供参考、不作方向依据。
2. 定绝对符号用**低速干净探**：小恒流(±120mA)、飞轮钳<400rpm,读 spd 加速方向 + 低速干净 gy + 落定 accel。
3. 起跳=蓄能急停,**蓄能转速从低 ramp**(欠冲落回自安全),把 apex 送到 θ≈0 低速再接管。
4. 平衡环 pitch 必须抗飞轮污染(陀螺积分/补偿)；但即便如此,动量预算墙仍在(见四)。
