# 001 — 屏动平衡系统（反作用轮倒立摆）

- 提出日期：2026-05-29
- 提出者：用户
- 状态：进行中

## 原始诉求
> 从头开始做这个项目（原有代码与板上原固件的功能视为不存在）。
> M5 框架、M5Atom / Roller 套件及相关库的 API 均已验证可用。
> 第一步：读取 ATOM 的 IMU，**换算成姿态角（俯仰角 pitch / 倾角 roll），不要原始数据**；
> 采样间隔沿用原工程的 50ms 保证实时性；读出后判断系统“可活动方向”与哪个角一致。

## 背景 / 动机
本体是一个**三棱柱**（截面近似等边三角形），立在底棱上、由内部**反作用轮**绕底棱长轴施加力矩，
做单轴自平衡（倒立摆）。这是“屏动平衡系统”。详见 [系统模型与标定](../system-model.md)。

## 目标与范围（分步推进）
- 第 1 步（已完成）：IMU → 姿态角、判定可动轴、标定平衡点。见 system-model.md。
- 不做（暂不涉及）：电机闭环控制、起摆策略 —— 在拿到电机协议并经授权前不驱动电机。

## 约束
- 硬件：M5Stack ATOM Matrix（ESP32 + MPU6886，无磁力计 → 无绝对 yaw）。
- 实时性：姿态采样 50ms（20Hz）。
- 安全：平衡/电机系统，未经授权不发任何会驱动电机的命令。

## 验收标准（按步）
- 第 1 步：能稳定输出 pitch/roll，明确可动轴、平衡点读数、工作区间。✅
