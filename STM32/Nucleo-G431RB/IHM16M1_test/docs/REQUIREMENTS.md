# REQUIREMENTS — 需求(做什么 / 为什么 / 怎样算完成)

> 规则:已确认的写成**肯定句并标日期**;未确认的放「待确认」区,**别当既定事实用**。

## 1. 目标(一句话)
用 **ST 电机控制套件(X-CUBE-MCSDK)** 标定一台**参数未知的散货三相 BLDC**,再生成并烧录 ST 官方 **FOC 固件**,用板载**霍尔传感器**做位置反馈,让电机在 FOC 下稳定运转、可调速。(2026-06-29 用户确认)

## 2. 硬件平台(已知,2026-06-29)
- **主控**:STM32 **Nucleo-G431RB** —— MCU = STM32G431RB,Cortex-M4F @170MHz,FPU + CORDIC + 高分辨率定时器,ST 主推电机控制 MCU。
- **扩展板**:**X-NUCLEO-IHM16M1** —— STSPIN830 三相 BLDC/PMSM 驱动板,支持单 / 三电阻电流采样,经 **Arduino UNO 排母排针叠插**到 G431RB。
- **电机**:一台**小型三相 BLDC**,**散货、卖家未给详细参数**(故需 Motor Profiler 标定);**自带霍尔传感器**,H1/H2/H3 + VCC + GND 已按扩展板说明书接入 IHM16M1。
- **关键**:`IHM16M1 + G431RB` = ST 官方电机控制套件 **P-NUCLEO-IHM03** 的板卡组合(官方原配电机为 gimbal 电机 GBM2804H-100T)→ MCSDK / Motor Profiler / Workbench 对此组合有官方支持;核心手册 **UM2538**(IHM03 pack FOC)/ **UM2415**(IHM16M1)(见 [`FACTS.md`](FACTS.md) F-7)。
  > 具体采样配置(单 / 三电阻跳线)、母线电压、引脚映射、霍尔接入引脚等**待核实写入 [`FACTS.md`](FACTS.md)**。

## 3. 功能需求
- [ ] **电机参数标定**:用 **Motor Profiler**(MCSDK 内)无感测出散货电机的 Rs / Ls / Ke / 极对数 / 额定电流等。
- [ ] **生成 FOC 固件**:把标定参数导入 **MC Workbench**,配置电流采样(本项目优先使用 IHM16M1 **默认三电阻**;UM2415/MCSDK 也支持 FOC 单电阻,但 Motor Profiler 不支持 single-shunt)、**霍尔作速度/位置反馈**、保护门限,生成工程。
- [ ] **编译 + 烧录**到 G431RB,电机在 FOC 下转起来。
- [ ] **闭环可控**:可设定 / 调节转速(目标速度跟随),霍尔反馈正常,启动可靠。

## 4. 非功能需求
- **安全保护**:过流 / 过压 / 欠压 / 堵转保护,故障即关断 PWM(默认安全,见 [`DESIGN.md`](DESIGN.md) §5)。
- 控制频率 / PWM 频率按 MCSDK 默认或电机适配;标定与首跑**电机空载、可急停**。

## 5. 验收标准(MVP)
- 散货电机经 Motor Profiler 标定出一组可用参数;
- 生成的 FOC 固件烧录后,电机能**稳定启动并在设定转速下运行**(霍尔反馈),用 MC Workbench / Motor Pilot 能监控转速 / 电流;
- 允许迭代调参,但要能**可靠转、不失步、不报故障停机**。

## 待确认 / 已知风险(查证 2026-06-29,2026-07-03 校正;见 [`FACTS.md`](FACTS.md) F-7~F-9/F-16/F-17)
- **采样配置**:UM2415 Rev 4 已确认 IHM16M1 **默认 FOC 三电阻**;FOC 单电阻也存在,所以"霍尔 FOC 必须三电阻/否则只能 6-step"是旧的过度简化。当前实物是否仍是默认三电阻**待用户按 [`IHM16M1_SHUNT_CONFIG.md`](IHM16M1_SHUNT_CONFIG.md) 看板确认**。本项目因 Motor Profiler 不支持 single-shunt,仍以三电阻为目标基线。
- **电机类型 / 转速**:是否 gimbal(高极对数、低速、低电流)?—— **Motor Profiler 对 gimbal / 低电流电机标定有已知坑**(逼近板子低压限、1-shunt 不稳),官方建议标定目标转速 **>1000 RPM**;散货电机能否顺利标定**需实测**,不保证一次成功。
- **霍尔低速 FOC 是已知难题**(电角度每 60° 跳变);若需极低速平滑运行,可能要额外算法。
- **供电**:电机母线电压 / 电流、外部电源方案待定(标定前必须确认在 IHM16M1 允许范围内)。
- **工具链**:需装 **X-CUBE-MCSDK 6.4.1**(本机 CubeMX 6.15 / G4 FW 1.6.1 / CubeCLT 1.18 已就位);由用户登录 st.com 安装(见 [`DECISIONS.md`](DECISIONS.md) #003)。
